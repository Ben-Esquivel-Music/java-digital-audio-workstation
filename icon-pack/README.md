# 🎵 AV Icon Pack

A professional SVG icon pack designed for Audio Visual applications — media players, DAWs (Digital Audio Workstations), podcast apps, and more.

## 🎨 Color Palette

| Color  | Hex       | Usage                          |
|--------|-----------|--------------------------------|
| White  | `#FFFFFF` | Details and neutral elements   |
| Green  | `#00E676` | Play, active, positive states  |
| Red    | `#FF1744` | Record, stop, alert states     |
| Purple | `#B388FF` | Creative, loop, effects        |
| Orange | `#FF9100` | Pause, shuffle, warm accents   |

### In HTML
```html

```

### In CSS
```css
.icon-play {
  background-image: url('icons/playback/play.svg');
  width: 48px;
  height: 48px;
}
```

### In React / JSX
```jsx

<PlayIcon width={48} height={48} />
```

## 📐 Design Principles

- **Consistent sizing**: All icons use a 48×48 viewBox
- **Transparent backgrounds**: Icons have no background shape — they integrate cleanly into any UI
- **Flat fills**: Clean, solid colors without gradients for a professional, modern appearance
- **Clean geometry**: Precise, minimal shapes for clarity at any size
- **Consistent stroke style**: Round caps and joins where applicable, with stroke widths varying by detail level (primarily 2px)
- **Color-coded categories**: Colors hint at function (green = go/active, red = stop/record, etc.)
