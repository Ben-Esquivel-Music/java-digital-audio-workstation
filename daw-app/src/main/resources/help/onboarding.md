# Onboarding Tour

The first time you launch the DAW, an onboarding tour highlights the main
controls — transport, mixer, arrangement, and browser — and shows a short help
text for each. You can dismiss the tour at any time and re-open it later from
the **Help** menu.

The tour visits topics in this order:

1. [Transport controls](transport)
2. [Arrangement view](arrangement)
3. [Mixer](mixer)
4. [Browser panel](browser)
