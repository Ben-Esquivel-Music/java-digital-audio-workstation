# Mixer

The mixer hosts one channel strip per track plus aux/master buses. Each strip
exposes:

- **Fader** — sets the channel level in dB.
- **Pan** — places the signal in the stereo (or surround) field.
- **Mute / Solo** — silence or isolate the channel.
- **Insert slots** — host plugins in series.
- **Sends** — route audio to aux buses.

Right-click a fader to reset to **0 dB**. Hold **Shift** while dragging a fader
for fine adjustments.
