# Transport Controls

The transport controls drive playback, recording, and looping for the entire
session.

## Buttons

- **Play** — start playback from the current playhead position.
- **Stop** — stop playback and (optionally) return to the start.
- **Record** — arm record on enabled tracks and start the transport.
- **Loop** — repeat playback between the locator markers.
- **Metronome** — toggle the click track.

## Keyboard shortcuts

```
Space      Play / Stop
R          Record
L          Toggle loop
M          Toggle metronome
```
